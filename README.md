# Discord Storefront Bot
## By: Ashsmith Khayrul

A discord.js v14 bot I made while I was in high school on commission. Complete with a fully functional ticket system, suggestions system, and review system. Built for any Discord server that sells products or services.

Check out my hosting service to get the bot online 24/7! Starting at just $1.80 a month, you can run your very own bot at [Uniplex Host](https://uniplex.xyz)! Feel free to open a ticket if you have any questions about the setup process or run into some trouble.

Step 1: Copy .env.example to .env and fill in your bot token and database credentials. If you're on a panel like Pterodactyl, set them as environment variables directly instead.

Step 2: Fill out the rest of the details in config.json including your server info, channel IDs, and role IDs.

Step 3: Start your server through the panel. Dependencies will install automatically and the required database table will be created on first startup.
